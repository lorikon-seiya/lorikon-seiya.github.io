# 🎥 北海道一周の旅｜Day 2

## 4月27日 新潟港 〜 北海道へ（小樽港）

# 旅途记录 4.27～4.28 / 旅の記録 4.27～4.28

## 4月27日 · 出发 / 出発

中午左右，坐上了从新潟出发的船。
新潟大概有70万人口，跟浜松差不多大的城市。没想到早上路上的车还挺多的。

昼ごろ、新潟からフェリーに乗った。
新潟は人口70万くらいで、浜松とだいたい同じくらいの規模の街だ。朝の道路が思ったより混んでいて、ちょっと意外だった。

!["新日本海フェリー"](/IMG_6774.jpeg)

## 出发前 / 乗る前に

去港口之前，顺路去了一趟 **マックスバリュ**。
想着船上的午饭和晚饭，买了一堆零食、泡面还有惣菜。东西买了不少。

港に行く前に、**マックスバリュ**に寄った。
船の中でのお昼とか夕ご飯のことを考えて、お菓子やカップ麺、惣菜をいろいろ買い込んだ。けっこうな量になった。

!["カレーヌードル"](/IMG_6778.jpeg)


## 船上 / 船の中で

刚上船的时候还挺兴奋的，到处转了好几圈。
但过了一会儿，新鲜感一过，就开始觉得好无聊了。

乗ったばかりのころはテンションが上がって、船の中をぐるぐる歩き回った。
でも少し経つと新鮮さも薄れて、だんだん暇になってきた。

!["船で撮った写真"](/IMG_6780.jpeg)


## 夜 / 夜

晚上10点多，船舱的灯关了。
躺下来之后也没睡太好——跟3年前从九州去大阪的那次比起来，感觉睡得差多了。

夜10時すぎに電気が消えた。
横になったけど、あんまりよく眠れなかった。3年前に九州から大阪に行ったときと比べると、全然眠れた感じがしない。

可能还是太兴奋了，睡不着。

たぶん、テンションが上がりすぎてたんだと思う。
