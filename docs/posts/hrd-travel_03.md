# 🎥 北海道一周の旅｜Day 2

## 4月28日 小樽 〜 千歳

# 旅途记录 4.28～4.29 / 旅の記録 4.28～4.29

## 抵达北海道 / 北海道に到着

29号凌晨4点半，船到了。地点是小樽。
下船之后第一件事就是去了**セイコーマート**——北海道才有的便利店。
天还完全没亮，早得有点离谱。

29日の朝4時半、船が小樽に着いた。
下船してすぐ向かったのは**セイコーマート**。北海道ならではのコンビニだ。
外はまだ真っ暗で、とんでもない時間だった。

!["小樽"](/IMG_6795.jpeg)

## 沿海公路 / 海沿いの道

从5号线拐上229号线，一路往前开。
北海道的路真的很宽，直线多，红绿灯又少，开起来特别爽。
风景也好，就这样舒舒服服地开了大约1个半小时，到了**積丹半島**。

5号線から229号線に入って、ひたすら走った。
北海道の道路って本当に広くて、直線が多くて、信号も少ない。運転してて気持ちよかった。
景色もよくて、気づいたら1時間半くらいで**積丹半島**に着いてた。

!["積丹半島"](/IMG_6808.jpeg)

## 積丹半島 / 積丹半島

北海道挺有名的观光地。但去得太早了，**神威岬**的门还锁着。
没办法就把车停在海边，在那里待了大约1个半小时。
钓鱼的人挺多的。海水透明度很高，从岬角那边看出去的海岸线也很漂亮。

北海道でも有名な観光地だ。でも早く着きすぎて、**神威岬**へのゲートがまだ閉まってた。
しょうがないので海岸に車を停めて、1時間半くらいぼーっとしてた。
釣りしてる人がたくさんいて、海の透明度がすごくて、岬から見える景色もきれいだった。

!["神威岬"](/IMG_6802.jpeg)

## 道の駅 → 羊蹄山 / 道の駅 → 羊蹄山

まだ時間があったので、そのまま**羊蹄山**方面へ。
途中の道の駅で揚げじゃがと揚げ芋を買って食べた。これが結構うまかった。
どこの道の駅だったか正確には覚えてないけど、地図見たらわかると思う。

时间还够，就继续往**羊蹄山**方向走。
途中在道の駅买了炸土豆和炸芋头，味道很不错。
具体哪个道の駅记不清了，不过地图上应该能查到。

!["羊蹄山"](/IMG_6812.jpeg)

## 洞爺湖 → 室蘭 → 千歳 → 札幌

そのあとは**洞爺湖**を通って**室蘭**へ、そこから**千歳**へ移動した。
夜は**札幌**まで行って、昔の知り合いと飯を食った。
場所が**麻生**で、なんか「麻生のオンナ」じゃんって思って、ちょっと懐かしい気持ちになった。

之后经过**洞爺湖**到**室蘭**，再从室蘭去了**千歳**。
晚上又跑了趟**札幌**，和以前认识的人一起吃了饭。
地点是**麻生**，脑子里自然就想起了那首《麻生のオンナ》，有点怀念。

!["白鳥大橋"](/IMG_6815.jpeg)
!["味噌ラーメン"](/IMG_6816.jpeg)

## 返回千歳 / 千歳に戻る

千歳の宿に戻ったのは夜の11時ごろ。
いっぱい走った一日だった。

回到千歳住的地方，已经是晚上11点了。
这一天，开了好长的路。
